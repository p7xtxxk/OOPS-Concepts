
# OOPs in Java

This repository is a comprehensive guide and code collection dedicated to __Object-Oriented Programming (OOP)__ in Java. It serves as a practical sandbox for developers to understand how to transition from procedural logic to robust, scalable, and reusable software design. 


__NB: Detailed explanations and conceptual notes are included directly within the Java source files as comments.__

## Roadmap

The following list outlines the progression of concepts covered in this repository, moving from foundational building blocks to advanced Object-Oriented design.

### Phase 1: Foundations of Classes and Objects

* Class Declaration and Syntax
* Object Instantiation and Memory Allocation
* Class Members: Fields (Attributes) and Methods
* The 'this' Keyword
* Access Modifiers: Private, Default, Protected, and Public

### Phase 2: Constructors and Initialization

* Default vs. Parameterized Constructors
* Constructor Overloading
* Copy Constructors
* Initialization Blocks (Static and Instance)

### Phase 3: Encapsulation and Data Hiding

* Getter and Setter Methods
* Data Validation within Setters
* Read-only and Write-only Classes
* Relationship between Encapsulation and Security

### Phase 4: Inheritance and Hierarchy

* Types of Inheritance (Single, Multilevel, Hierarchical)
* The 'super' Keyword (Accessing Parent Members and Constructors)
* Method Overriding and the @Override Annotation
* Preventing Inheritance with the 'final' Keyword

### Phase 5: Polymorphism

* Compile-time Polymorphism (Method Overloading)
* Runtime Polymorphism (Dynamic Method Dispatch)
* Upcasting and Downcasting of Objects
* Covariant Return Types

### Phase 6: Abstraction

* Abstract Classes and Abstract Methods
* Interface Declaration and Implementation
* Multiple Inheritance via Interfaces
* Default and Static Methods in Interfaces (Java 8+)
* Private Methods in Interfaces (Java 9+)

### Phase 7: Advanced OOP Relationships

* Association, Aggregation, and Composition
* Inner Classes (Nested, Inner, Local, and Anonymous)
* Records and Sealed Classes (Modern Java Features)
